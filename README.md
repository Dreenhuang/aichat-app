# aichat-app

AI Chat Application - A modern chat app powered by AI

## 技术栈

- React 18 + TypeScript
- Vite
- Tailwind CSS

## 开始开发

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```

## 项目结构

```
aichat-app/
├── src/
│   ├── components/     # React 组件
│   ├── pages/         # 页面
│   ├── hooks/         # 自定义 Hooks
│   ├── services/      # API 服务
│   └── utils/         # 工具函数
├── public/            # 静态资源
└── ...
```
